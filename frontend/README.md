# AI Chat Interface

Based on the Private Chatbot AI Chat Interface Design Research Report.

## Tech Stack
- Vue 3
- Vite
- Tailwind CSS
- FontAwesome

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Features
- **Tech Modern Style**: Blue/White/Gray color scheme.
- **Components**:
  - Sidebar with history.
  - Chat Interface with multimedia input options.
  - Message Bubbles (User/AI distinct styles).
  - Typing indicators.
